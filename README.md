# Dear Body

**Dear Body** is a poetic, blog-based web application where users can write anonymous
letters to their body or specific body parts.  
The project creates a soft and personal digital space for reflection, emotional
release, and intimacy.

This project was developed as a **solo project** for **Creative Code Lab 2 (CCL2)**
